Troubleshoot the Wiz- integration

Error

Resolution

When running the remediation in Wiz, it fails with a 401 error.

Make sure the authentication settings for the flow are saved, and the authentication is configured correctly in the Wiz Integration.

Unauthorized in flows for OAuth2.0 Authentication in Identity Flows.

  • Verify the service user credentials.

  • Verify that the web app ID in the token URL matches your web app.

The integration is executed and the Identity flow is triggered, but Policy creation fails with a 401 error.

  • Verify that the service user has permissions on the web app.

  • Verify that the web app has the correct scope configured.
