URL in On-demand access flows

On-demand access flows for Service Now and Slack, as well as any custom flows that include components, need to be updated due to internal infrastructure changes.

When did this change take place?

July 7, 2024

What changed?

The URL changed from

{tenant_name}-sca.cyberark.cloud

to

{tenant_name}.sca.cyberark.cloud

What you need to do

flows for that were downloaded and configured before July 7, 2024 need to be updated. This includes flows for ServiceNow or Slack integrations, as well as any custom flows you have built.

To prevent degradation in functionality, you must update the URL by August 31, 2024.

After this date, if your flows still have the original configuration:

  • will no longer send email notifications when users request access

  • will no longer create policy for approved requests

  • will no longer send email notification for approved or denied requests
