policy APIs

This topic describes the API operations that you can perform for policies. For example, you can programmatically create, update, or delete a policy.

There are two versions of the Create Policy, Edit Policy, and Get Policy APIs; version 1.0 and version 2.0. Use the X-API-Version parameter to indicate which version you want to run. The default value is 1.0.
