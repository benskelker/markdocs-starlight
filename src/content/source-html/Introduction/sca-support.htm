Known limitations

This section describes known limitations with .

Search

When performing a search in , consider the following limitations:

  • If your search includes a hyphen (-), the search will return all results that include any of the following:

    • The term before the hyphen

    • The term after the hyphen

    • The hyphen itself

    Searching for "Test-Policy" returns any results that include:

    • Test

    • Policy

    • -

  • The policy table search capability is limited to these fields:

    • Policy name

    • Policy description

    • Related organization ID

    • Related account - workspace ID

    • Related account name

    • Related user name / user display name
