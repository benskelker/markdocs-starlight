Google Cloud permissions used by service account

This document describes the Google Cloud permissions needed by 's to implement zero standing privileges for identities in your organization.

Permissions

The following table lists the permissions needed for the service account, which is deployed in a project in the Google Cloud organization based on the configuration that was defined during onboarding.
